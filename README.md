## Stack

- HTML5 + CSS3 (sem frameworks, sem build step)
- JavaScript vanilla (tema claro/escuro, animações de scroll)
- Fontes: [Fraunces](https://fonts.google.com/specimen/Fraunces),
  [Inter](https://fonts.google.com/specimen/Inter),
  [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono)

## Estrutura

```
index.html
assets/
  css/style.css
  js/main.js
```

## Rodar localmente

Basta abrir `index.html` no navegador, ou servir a pasta com qualquer
servidor estático, por exemplo:

```bash
npx serve .
```

## Publicação (GitHub Pages)

Este repositório é publicado automaticamente em `https://cleytontome.github.io`
a partir da branch `main` (Settings → Pages → Source: `main` / `/root`).
